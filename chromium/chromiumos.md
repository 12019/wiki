### Getting to a command prompt
* You can get to VT-2 by pressing: ```[ Ctrl ] [ Alt ] [ => ]``` 
 * the [ => ] key is the right-arrow key just above the number 3 on your keyboard.
* Get back to the browser. Just press: ```[ Ctrl ] [ Alt ] [ <= ]```
